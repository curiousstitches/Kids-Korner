# data/

Buddy's content libraries - all safe to edit with Notepad.

- `responses.json` — spare offline reply lines by age group.
- `language/vocabulary.json` — teaching words with emoji and kid-friendly meanings, by age.
- `stories/stories.json` — short read-aloud stories by age, each with learning words.

Add your own entries by copying an existing line and keeping the same format.
