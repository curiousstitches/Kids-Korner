# language/

Vocabulary lists Buddy uses to teach words.

- `vocabulary.json` — words grouped by age tier (`3`, `6`, `12`, ...). Each word has an emoji and a kid-friendly meaning, e.g. `{"word": "curious", "emoji": "🤔", "meaning": "wanting to learn new things"}`.

Add your own words by copying an existing line and editing it — keep the same format and matching age group.
