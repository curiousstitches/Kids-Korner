# Kids Korner - AI Chat with Buddy

A friendly AI chat application designed for kids 3+ and adults. Features real AI responses, human-like conversation, and a caring personality named Buddy.

## Features

- **Real AI responses** - Uses Hugging Face models (no API key needed!)
- **Age-specific responses**: 3+, 6+, 12+, and Adult modes
- **Live mic chat**: Click 🎤 for continuous voice conversation
- **Auto voice response**: Buddy speaks back automatically
- **Memory/learning**: Remembers name, interests, favorites
- **Caring personality**: Buddy expresses love, never says "I'm just an AI"

## Getting Started

### Full AI Mode (Recommended)
1. Install Python 3.x and Flask: `pip install flask requests`
2. Run `python app.py`
3. Open `http://localhost:8081` in your browser
4. Buddy uses real AI responses!

### Simple Mode (No AI)
- Just open `index.html` directly - uses pre-written responses

## Meet Buddy - Your AI Friend!

Buddy is a friendly AI who loves to chat with you! He has a cheerful personality and always tries to make conversations fun and engaging. Buddy's catchphrases include:
- "I love that!"
- "That makes me happy!"
- "You're so cool!"
- "You light up my day!"
- "That's awesome!"

Buddy will:
- Remember your name after you tell him
- Express affection appropriately for your age group
- Check in regularly to say he loves talking with you
- Respond with enthusiasm to everything you share

## Voice Chat

1. Click the 🎤 button to start voice chat
2. Buddy keeps listening continuously
3. When you speak, Buddy responds and speaks back automatically
4. Click 🔴 to stop listening

## Safe & Appropriate

- All responses are carefully crafted to be appropriate for each age group
- Buddy expresses affection in safe, non-physical ways
- No personal data is collected
- Works with free Hugging Face AI models
