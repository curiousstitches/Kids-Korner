# 🔐 Buddy-Vault/

Your personal API key vault - a standalone mini-app.

## What it is
`vault.html` - double-click it, no install, no server, no internet needed. It's the control room for every key the Buddy apps can use.

## What it does
- **Every category covered**: Chat AI, Picture Generation, Voice, Texting - each pre-loaded with free providers, sign-up links, endpoints, and the exact env var names the apps read
- **Add / edit / remove / copy** any key with one click; add your own custom providers and fields
- **Set the active provider** per category (green border = active) and switch anytime
- **📋 Copy keys.local.bat** - one click builds the exact content for your private keys file
- **📋 Copy .env block** - same thing formatted for Render's Environment screen
- **💾 Export / 📂 Import** - back up the whole vault to a file and restore it anywhere
- **🔒 Optional passphrase** - real AES encryption; without the passphrase the vault won't open

## Where keys live
Only in this browser, on this computer. Nothing is uploaded anywhere, ever. Sharing the app folder does NOT share your vault. Exported backup files DO contain keys - keep them private.

## No keys? No problem
Providers marked **NO KEY NEEDED** (Pollinations drawing, browser voices, Ollama local AI) work the moment you download the apps - that's what makes the folder safe and useful to share with others, who can then add their own keys.
